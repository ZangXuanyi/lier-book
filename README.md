# 《重构仙途》

如你所见，这是一本小说！

其实在GitHub上发布小说似乎不是一件常见的事，但我觉得这是一个不错的方式来分享我的作品。主要原因自然是其超高的自由度，不必面对任何审查和删贴的风险，以及可以随时更新和修改内容的便利性。

毕竟我的小说一开始的目的便是一时兴起、取悦自己的，完全不考虑商业化的因素——当然之后也确实证明了其商业化是失败的了。

但故事是个好故事，我不愿意因为一些形式上的问题而放弃它，于是就有了把它放在GitHub上的想法。

我这种不习惯于写传统风格网文的人，采用这种形式反而如鱼得水——我甚至不必担心分章节的问题，一个 `vspace` 就能解决几乎全部的问题。

这多是一件好事啊！我可以随时添加新的章节，修正之前的错误，或在原有文字基础上开枝散叶。这种互动性是传统出版方式无法比拟的，甚至常规网文平台都无法提供这样的灵活性。毕竟几乎所有的网文平台都不支持MarkDown！！！

当然了，这也意味着我需要自己承担所有的编辑和排版工作，但有了LaTeX以后，这也不是一件难事。

## 关于本文

- 本文的版权归作者所有，未经授权禁止转载。内容纯属虚构，如有雷同，纯属巧合。
- 本文的更新频率不定，可能会根据作者的创作进度和灵感而有所调整。
- 本文允许读者共创！
    + 关于读者共创，希望每个对现有剧情有意见、希望自己改写的读者都能在评论区留言，或者直接提交Pull Request来修改文本内容。无论是剧情发展、角色设定还是世界观构建，只要你有想法，都欢迎参与到这个创作过程中来！当然了，所有的修改都会经过作者的审核，以确保故事的连贯性和质量。
    + `main` 分支是作者原汁原味的版本，不会被任何PR修改；而 `master` 分支则是读者共创的版本（如改编剧情、同人、番外等），包含了所有经过审核的读者修改内容。这样一来，读者们既可以看到原版故事，也可以体验到由社区共同创作的版本，享受不同的阅读体验！
